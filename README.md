**PORTUGUESE SKULL ALERT IN GALICIA**

Real-time monitoring of jellyfish hazards using the Open Weather API, InfluxDB, and Grafana.

Tech stack: Python 3, InfluxDB 2.x, Grafana.

Monitors 12 locations via coordinates.
